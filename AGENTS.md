# Arahan Ejen Projek (Workflow Tetap)

## Bahasa
- Semua balasan kepada pengguna hendaklah dalam Bahasa Melayu.

## Aliran Kerja Selepas Selesai Tugasan
1. Jalankan `npm run build`.
2. Jika berjaya, terus `git add -A`.
3. Buat `git commit` dengan mesej yang sesuai.
4. Terus `git push` ke branch semasa tanpa bertanya semula.

## Nota
- Jika `npm run build` gagal, maklumkan ralat dan baiki dahulu sebelum commit/push.
- Jika tiada perubahan fail, maklumkan status tanpa commit kosong.
